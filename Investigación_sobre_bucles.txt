# Resumen: Investigación sobre Bucles en Programación

## 1. Bucles: Controlados por Contador vs Condición

**Bucle por Contador (`for`):**
- Número predeterminado de iteraciones
- Se conoce de antemano cuántas veces se ejecutará
- Ejemplo: `for i in range(1, 6): print(i)`
- Casos de uso: Procesar listas, rangos numéricos definidos

**Bucle por Condición (`while`):**
- Se ejecuta mientras la condición sea verdadera
- Número de iteraciones desconocido al inicio
- Ejemplo: `while intento != password: intento = input()`
- Casos de uso: Validaciones, esperar eventos, entrada de usuario


## 2. Ejemplos Cotidianos

**Bucle por Contador:**
- Hornear 4 tandas de galletas (cantidad conocida)
- Hacer 3 series de 10 flexiones
- Enviar invitaciones a lista de amigos

**Bucle por Condición:**
- Cajero automático: retirar hasta que saldo < 100
- Juego de adivinanza: hasta acertar el número
- Llenar taza: hasta alcanzar capacidad máxima

## 3. Cuándo Usar `while` en Lugar de `for`

**Usa `while` cuando:**
- No conoces el número exacto de iteraciones
- La condición depende de estado dinámico
- Necesitas validar entrada del usuario
- Esperas un evento externo

**Usa `for` cuando:**
- Conoces exactamente cuántas veces iterar
- Necesitas recorrer una secuencia/lista
- Trabajas con rangos numéricos específicos


## 4. Bucles Infinitos

**Definición:** Ciclo que nunca termina porque su condición nunca es falsa.

**Ejemplo:**
```python
# ❌ ERROR: Olvidar actualizar variable
contador = 0
while contador < 10:
    print(contador)  # Falta: contador += 1
```

**Cómo Prevenir:**
1. Asegurar que la condición pueda ser falsa: `contador += 1`
2. Usar límite de seguridad: `if intentos > max: break`
3. Verificar lógica: incrementar/decrementar correctamente
4. Cuidado con floats: usar `<` en lugar de `!=`

**Cómo Detectar:**
- Programa no responde
- CPU al 100%
- Mismo mensaje se repite infinitamente
- **Solución:** Presionar `Ctrl + C` para detener


## 5. Sentencias `break` y `continue`

**`break` - Termina el bucle completamente:**
```python
# Buscar número en lista
for num in [4, 7, 12, 9, 3]:
    if num == 9:
        print("¡Encontrado!")
        break  # Sale del bucle
```
Usos: Encontrar elemento, salir de menú, límite de intentos

**`continue` - Salta a la siguiente iteración:**
```python
# Imprimir solo impares
for num in range(1, 11):
    if num % 2 == 0:
        continue  # Salta pares
    print(num)  # Solo imprime 1, 3, 5, 7, 9
```
Usos: Filtrar elementos, validar entrada, omitir casos específicos

**Diferencia:**
- `break` → termina TODO el bucle
- `continue` → salta SOLO esa iteración


## 6. Error Lógico Más Común en `while`

**❌ ERROR #1: Olvidar Actualizar Variable de Control**
```python
# ERROR: Bucle infinito
contador = 0
while contador < 5:
    print(contador)  # Falta: contador += 1
```

**Otros Errores Comunes:**

2. **Condición inalcanzable:** `numero = 10; while numero < 10:` (ya es falsa)
3. **Dirección incorrecta:** `while x > 0: x += 1` (aumenta en lugar de disminuir)
4. **Condición siempre verdadera:** Mayúsculas/minúsculas no coinciden
5. **Usar `=` en lugar de `==`:** Asignación vs comparación

**✅ Mejores Prácticas:**
- Planifica: condición inicial, de parada y cómo cambia la variable
- Usa nombres descriptivos: `intentos_restantes` vs `x`
- Agrega validaciones de seguridad durante desarrollo
- Prueba con valores límite
- Siempre actualiza la variable de control: **`contador += 1`**

## Resumen

| Concepto | Descripción | Cuándo usar |
|----------|-------------|-------------|
| **Bucle por contador** | Iteraciones predeterminadas | Número exacto de repeticiones |
| **Bucle por condición** | Mientras condición verdadera | Estado dinámico |
| **`for`** | Secuencias o rangos | Listas, cadenas, rangos definidos |
| **`while`** | Condición verdadera | Validaciones, eventos externos |
| **`break`** | Termina el bucle | Salir anticipadamente |
| **`continue`** | Salta iteración | Omitir casos específicos |
| **Bucle infinito** | Nunca termina | ¡EVITAR! Actualizar variable |

---

**Puntos Clave:**
- `for`: sé cuántas veces → `while`: no sé cuántas veces
- Siempre actualizar variable de control en `while`
- `break` sale, `continue` salta
- `Ctrl + C` detiene bucle infinito
- Validar entrada y límites de seguridad previenen errores
